# AuctionSystem
EECS3311 Assignment

1. Create a TomCat V9.0 Server for the AuctionSystem Project (Do not start it yet)
2. Run the "populateDatabase.java" file to create some auctions in the database (Optionally add your own examples to the file)
3. Start the server
4. On a browser, head to localhost:8080/AuctionSystem/AuctionListServlet
5. Fill in the fields to bid on an item, and press the place bid button to confirm.
6. Refresh the page to update auction status if needed
